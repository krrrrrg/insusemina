## 작업 의도

기존 랜딩페이지(index.html, contact.html)의 카피, 섹션 순서, 콘텐츠를 클라이언트 피드백에 맞춰 전면 갱신한다. 신규 페이지 추가는 없으며, 기존 구조 안에서 텍스트/이미지/순서를 교체하는 작업이다.

---

## 목표

- 클라이언트가 회의(03/10)에서 요청한 카피 변경, 섹션 재배치, 콘텐츠 교체를 정확히 반영한다.
- 디자인 톤(토스 스타일, Navy + 토스블루)과 기존 레이아웃 일체감을 유지한다.
- 타겟 사용자(보험 영업인)가 공감할 수 있는 흐름으로 섹션을 재구성한다.

## 요구사항

| 서브카피 | "성과를 가로막는 지점, 10년간 온/오프라인 DB만 연구한 전문가가 정확하게 해결해 드립니다." |

### 2. Be your signature 섹션 - 카피 변경

| 항목 | 내용 |
|------|------|
| 타이틀 | "1:1 맞춤 진단으로 바로 적용가능한 솔루션" |
| 설명 | "진부한 스크립트, 녹취 공유로 끝내지 않습니다. 본인에게 맞는 시장, 성과가 나오는 프로세스를 구성해 드립니다." |

### 3. 통계 수치 변경

| 항목 | 값 |
|------|-----|
| 성공률 | 3배 |
| 체결률 | 50% |
| 수익 | 7배 (100만 → 700만) |
| 순수익 | 800만원 |

### 4. 섹션 순서 변경 + Pain Points 내용 교체

- Pain Points 섹션을 Be your signature 섹션 **바로 뒤**로 이동한다.
- Pain Points 내용을 아래 4개로 전면 교체:
  1. "바빠요", "다음에요" 숨막히는 단박거절. DB가 들어와도 전화 거는게 스트레스입니다.
  2. 큰 맘 먹고 샀는데, "이거 왜 하는 거예요?" 묻는 말에 회의감만 쌓여 갑니다.
  3. 운 좋게 약속을 잡아도 첫인사부터 어떻게 상담을 이끌고 나가야 2차가 잡히는지 감이 안와요.
  4. 소개는 대체 어떻게 받는 거죠? DB로 영업 잘하는 사람들이 신기합니다.

### 5. Solution 섹션 변경

| 항목 | 내용 |
|------|------|
| 카피 | "DB 시장에 대한 이해와 나에게 맞는 선택, 성공하는 영업 전략까지." |
| 추가 요소 | 영상 삽입 영역 (숏폼 1분대 - 과정 소개/의미/얻어갈 것) |

### 6. Reviews 섹션 - 후기 3개 전면 교체

1. **DB 최적화**: "제가 쓰고 있는 DB에 대한 객관적인 진단, DB종류부터 선택까지 가장 중요한 부분을 컨설팅 받았습니다."
2. **약속이 잡히는 TA**: "1:1 맞춤 TA 전략을 사용하니 상담이 잡히기 시작했어요. TA를 두려워하지 않게 됐습니다. 처음으로 3방 달성했어요!"
3. **소개가 나오는 상담**: "계약은 물론 고객님의 소중한 가족, 지인까지 자연스럽게 이어지는 프로세스를 배웠습니다. 1건으로 5건 6건의 효과를 낼 수 있어요!"
